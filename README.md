Raqeval is a high-performance **math expression parser and evaluator** written in 100% pure Rust. Designed for speed and minimalism, raqeval features zero external dependencies, making it an ideal choice for performance-critical environments like game engines, real-time simulations, and embedded systems. 

## Features

- ✅ **Zero Dependencies**: Keeps your dependency graph clean and compile times fast.
- ✅ **Rust-Native**: Built from the ground up to be small and portable.

## WARNING 

This library was created primarily for **learning purposes**. It is currently work-in-progress (WIP) and may not yet be the most efficient or feature-complete implementation. **It is not recommended for production use quite yet**.
